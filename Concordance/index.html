Concordance

A concordance is a list of key words used in a body of text. Usually they list the each instance of the word in its immediate context.

This page demonstrates a simple concordance by counting the number of occurrences of all words in a body of text. Drag and drop any files to see a list of all words sorted by counts. File issues and view source on github. Once you have counted the words you can click on Create Table and a .csv file will be generated.

This file is based on Daniel Shiffman's example from 'Programming from A to Z'. See https://shiffman.net/a2z/text-analysis/ for a tutorial on this code and other examples

See http://textalyser.net for more in depth analysis
